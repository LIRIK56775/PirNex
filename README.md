🏴‍☠️ PirNex — Nexus Mods Auto Downloader

PirNex — это легкий и эффективный инструмент для автоматизации загрузки модов с сайта Nexus Mods. Больше не нужно ждать таймеры и вручную подтверждать каждый файл — просто вставьте список ссылок и занимайтесь своими делами.



✨ Основные возможности

Массовая загрузка: Добавляйте десятки ссылок одновременно.



Умная очередь: Последовательная обработка ссылок без конфликтов.



Автономный профиль: Бот сохраняет авторизацию в отдельной папке — вам не нужно входить в аккаунт каждый раз.



GUI на CustomTkinter: Современный и понятный интерфейс.



🚀 Как пользоваться (для пользователей EXE)

Скачайте последнюю версию в разделе Releases.



Распакуйте архив в любую удобную папку.



Запустите программу. Рядом появятся папки downloads и nexus\_bot\_profile.



Важно при первом запуске:



Вставьте любую ссылку и нажмите кнопку добавления.



В открывшемся окне браузера войдите в свой аккаунт Nexus Mods.



Убедитесь, что в настройках Chrome включена "Стандартная защита" (Settings -> Security).



Нажмите ОК в окне уведомления бота.



Готово! Теперь бот будет качать всё автоматически, авторизация больше не потребуется.



🛠 Инструкция для разработчиков (Python)

Если вы хотите запустить проект из исходного кода:



Клонируйте репозиторий: git clone https://github.com/LIRIK56775/PirNex.git



Установите зависимости: pip install customtkinter undetected-chromedriver



Запустите: python main.py



📦 Сборка в EXE

Для самостоятельной сборки используйте PyInstaller: python -m PyInstaller --noconfirm --onefile --windowed --name "PirNex" --collect-all undetected\_chromedriver main.py



🛡 Безопасность и конфиденциальность

Профиль: Все данные авторизации хранятся локально в папке nexus\_bot\_profile. Никогда не передавайте эту папку третьим лицам.



Open Source: Исходный код полностью открыт. Вы можете убедиться, что программа не передает ваши данные и взаимодействует только с официальным сайтом Nexus Mods.
