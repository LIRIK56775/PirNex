🏴‍☠️ PirNex — Nexus Mods Auto Downloader

PirNex is a lightweight and effective tool for automating mod downloads from the Nexus Mods website. No more waiting for timers or manually confirming each file—just paste a list of links and go about your business.



✨ Key Features

Bulk Download: Add dozens of links at once.



Smart Queue: Process links sequentially without conflicts.



Standalone Profile: The bot saves your login information in a separate folder, eliminating the need to log in each time.



CustomTkinter GUI: Modern and intuitive interface.



🚀 How to Use (for EXE users)

Download the latest version in the Releases section.



The bot uses Google Chrome version 145.



While the bot is running, you can use any browser other than Google Chrome to select mods.



Unzip the archive to any convenient folder.



Launch the program. The downloads and nexus\_bot\_profile folders will appear next to each other.



NEVER share the nexus\_bot\_profile folder with anyone.



Important on first launch:



Paste any link and click the add button.



In the browser window that opens, log in to your Nexus Mods account.



Click OK in the bot notification window only after logging in.



Paste each link in the input field on a new line.



Done! The bot will now download everything automatically; authorization is no longer required.



!!!If the bot suddenly stops downloading mods, try deleting everything from the nexus\_bot\_profile folder.
